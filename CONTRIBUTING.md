# Contributing to Django IDE Extension

Thank you for your interest in helping improve the **Django IDE Extension**! While the main language server and core client repository are currently private, this repository serves as the public face for documentation, issue tracking, and community discussions.

We welcome contributions of all types:
- **Reporting Bugs:** Submitting clear issue reports when the extension doesn't behave as expected.
- **Feature Requests:** Suggesting new features or improvements.
- **Documentation:** Improving this public documentation or website.

## How to Report issues

To report a bug or request a feature:
1. Search the [GitHub Issues](https://github.com/your-username/django-ide-extension/issues) page to check if it has already been reported.
2. If not, open a new issue using a descriptive title and detailed template.
3. For bugs, please include:
   - Your VS Code version.
   - Your Django version and Python version.
   - A minimal reproducible example (e.g., Python view code or Django HTML template snippet).
   - Any LSP logs (can be found in the VS Code Output panel under "Django Language Server").

## Document updates

If you notice spelling errors, broken links, or want to expand on the documentation/landing page:
1. Fork this repository.
2. Create a new branch for your changes: `git checkout -b update-documentation`.
3. Commit and push your changes.
4. Open a Pull Request pointing to the `main` branch.
